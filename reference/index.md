# Package index

## CPI adjustment

Adjust monetary values using Consumer Price Index data

- [`adjust_inflation()`](https://charlescoverdale.github.io/inflateR/reference/adjust_inflation.md)
  : Adjust a historical monetary value for inflation
- [`historical_value()`](https://charlescoverdale.github.io/inflateR/reference/historical_value.md)
  : Convert a present-day value to its historical equivalent

## GDP deflator adjustment

Adjust monetary values using GDP deflator data

- [`adjust_real()`](https://charlescoverdale.github.io/inflateR/reference/adjust_real.md)
  : Adjust a monetary value using a GDP deflator
- [`historical_real()`](https://charlescoverdale.github.io/inflateR/reference/historical_real.md)
  : Convert a value to its historical equivalent using a GDP deflator
